ubuntu挂在8T级别硬盘

Last updated on Mar 14, 2022 1 min read programme, 技术
Reading time: 2 minutes and 7 seconds with 467 words. 本文总阅读量次
  • 普通情况下，挂载小于等于2T磁盘
  • 挂在8T硬盘的方法

Ubuntu挂在不同容量硬盘有所区别，使用命令有所不同，小容量用fdisk,大容量用parted

普通情况下，挂载小于等于2T磁盘

fdisk -l       ##（查看当前分区的情况）
fdisk /dev/vdd
m
p
n
p
1
空格
空格
w
mkfs -text4 /dev/vdb1  ## 初始化磁盘
vim /etc/fstab  ##打开fstab写下下面内容，其中/minio/data就是挂在的路径，后面内容默认即可
/dev/vdc1 /minio/data              ext4     defaults        0 0
mount -a

挂在8T硬盘的方法

1.parted /dev/vdc  ## 一般大于2T，需要用parted命令  fdisk -l 可以查看所有设备信息
2.print ##（查看一下）
3.mklabel gpt ##（设置分区类型为gpt）
unit TB  ##(设置单位为TB)
4.mkpart extended 0% 100% ## 划分分区大小， 建立拓展分区。表示所有扇区都用于当前分区）
##mkpart primary 0 3 ## 划分一个主分区,大小为3TB，开始是0，结束是3  目前清能院得是 mkpart primary 0  表示8T)
5.print ##(显示设置的分区大小）
6.quit
##Information: You may need to update /etc/fstab.
7.mkfs.ext4 /dev/vdc  ##格式化新硬盘 与2T以下有所不同   <-----------------------
8. vim /etc/fstab  ##写入一行信息：(涉及到硬盘和文件夹的关系)

```
/dev/sdb         /tempo     ext4     defaults        0 0    <-------------/tempo代表挂载的目录(mkdir /tempo)
```

9. mount  /dec/sdb  ## <------------或者mount -a  如果有多快盘,可以全部挂载上
