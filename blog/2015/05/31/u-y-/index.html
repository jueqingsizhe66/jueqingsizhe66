数据段

注意gnuplot处理数据文件按照空行划分index部分。所以下面的部分总共有四个index，调用方式是

  • plot “文件名” index 0 第一部分文件信息的打印
  • plot “文件名” index 1
  • plot “文件名” index 2
  • plot “文件名” index 3
